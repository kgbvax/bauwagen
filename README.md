# bauwagen
Collection of bits that helps me run the DL9ET shack:

* ACOM Mon
A service to monitor and control ACOM 600S and 1200S solid-state amplifiers via their RS-232 serial interface. It decodes the amplifier's proprietary telemetry protocol and integrates seamlessly with Home Assistant via MQTT Auto Discovery.
